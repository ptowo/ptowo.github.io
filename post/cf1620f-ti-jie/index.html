CF1620F 题解

2022-05-17 6 min read

因为是二分图，根据连边规则，注意到，当数组中不存在一个长度大于等于 333 的递减子数列是合法的。

于是问题转化为构造一个数组 aaa, 满足 aaa 中没有长度超过 222 的严格下降子序列。

然后我就只会一个 n3n^3n3 的暴力了，咍咍。

然后学习了一下网上的题解。

假设 aja_jaj​ 满足 ppp 性质，当且仅当存在下标 i<ji < ji<j 使得 ai>aja_i>a_jai​>aj​。

我们设立状态 dpi,u,vdp_{i,u,v}dpi,u,v​ 表示当前确定了序列前 iii 位，其中最大的值为 uuu，满足 ppp 性质的最大数为 vvv（如果不存在，则取 −∞-\infty−∞ ）。

显然 u>vu>vu>v，假设 aaa 的第 i+1i+1i+1 位取 www。

若 w>uw > uw>u，则 dpi,u,v→dpi+1,w,vdp_{i,u,v} \rightarrow dp_{i+1,w,v}dpi,u,v​→dpi+1,w,v​。

若 v<w<uv < w < uv<w<u，则 dpi,u,v→dpi+1,u,wdp_{i,u,v} \rightarrow dp_{i+1,u,w}dpi,u,v​→dpi+1,u,w​。

如果 w<vw < vw<v 的话，状态转移一定不合法，因为 vvv 满足 ppp 性质，如果右边还有一个小于 vvv 的数，那么这三个数将会构成一个长度为 333 的严格下降子序列。

然后注意到，合法的状态经过转移，要么 uuu 变大，要么 vvv 变大。

对于同一下标的两个状态，若 u1≤u2,v1≤v2u_1\leq u_2,v_1 \leq v_2u1​≤u2​,v1​≤v2​，则称状态 (i,u1,v1)(i,u_1,v_1)(i,u1​,v1​) 优于 (i,u2,v2)(i,u_2,v_2)(i,u2​,v2​)。

如果最劣的状态能转移，那么所有状态都能转移。如果最优的状态不能转移，那么所有状态都不能转移。

于是只保留最优状态，通过归纳法证明每个下标对应的不同状态数不会超过 222。

采用数学归纳法进行证明：

一开始的两个状态为 (1,−p1,−∞),(1,p1,−∞)(1,-p_1,- \infty),(1,p_1,-\infty)(1,−p1​,−∞),(1,p1​,−∞)，但是由于前面一个状态更优，于是后面的被去掉，最后就只有一个状态。

如果下标 iii 有 000 个状态，那么无法转移，于是以后都只有 000 个状态。

如果下标 iii 具有 111 个状态，两种不同取值，则会带来 222 中状态。

如果下标 iii 有 222 个状态 (i,u1,v1),(i,u2,v2)(i,u_1,v_1),(i,u_2,v_2)(i,u1​,v1​),(i,u2​,v2​)，设 u1>u2u_1 > u_2u1​>u2​，那么 v1<v2v_1 < v_2v1​<v2​，然后分析可得 u1>u2>v2>v1u_1>u_2>v_2>v_1u1​>u2​>v2​>v1​。

然后枚举 i+1i+1i+1 的取值 www 的相对大小分析可以得到还是只会有 222 个状态。

那么每个下标对应状态数不会超过 222，然后每次转移，之后只保留最优的状态就好了。

}
const int N = 2e6;
int T, n, p[N], ans[N][2];
vector <int> tp, q;
void ins(int id,int k) {
  int x = k ? p[id] : -p[id];
  int i = upper_bound(q.begin(), q.end(), x) - q.begin(); if (!i) return;
  int j = q.size() - i;
  tp.emplace_back(min(q[j], x)); tp.emplace_back(max(q[j], x));
  if (abs(q[q.size() - 1 - j]) == p[id - 1]) ans[id][k] = q[q.size() - 1- j] > 0;
  if (abs(q[j]) == p[id - 1]) ans[id][k] = q[j] > 0;
  return;
}
bool check() {
  q.clear(); if (tp.empty()) return false;
  if (tp[0] <= tp[2] && tp[1] <= tp[3]) { tp.pop_back(); tp.pop_back(); }
  sort(tp.begin(), tp.end()); q = tp; return true;
}
void solve() {
  n = read();
  for (int i = 1; i <= n; i++) p[i] = read(); q.clear(); tp.clear();
  for (int i = 1; i <= n; i++) {
    tp.clear();
    if (q.empty()) { q.push_back(- n - 1); q.push_back(-p[i]); continue; }
    ins(i, 0); ins(i, 1);

    if (abs(v) == p[n]) {

    }
  }
}

  T = read(); while (T--) solve();
  return 0;
